# 小说情节设计框架

## 适用场景
当用户需要创作小说、故事、剧本等叙事性内容时应用此框架。

## 情节设计四要素
1. **冲突**：主角面临的内外部矛盾，驱动故事前进的核心动力
2. **发展**：矛盾逐步升级，主角经历考验和成长
3. **高潮**：矛盾集中爆发，主角做出关键抉择
4. **结局**：矛盾解决，主题升华

## 推荐工作流
1. **构思阶段**：明确主题、类型、目标读者
2. **人物设计**：主角、配角、反派的性格和动机
3. **大纲设计**：用三幕结构或起承转合规划章节
4. **场景细化**：每个场景的目的、冲突、推进
5. **正文写作**：注重节奏和张力控制

## 写作技巧
- 每个场景必须推进情节或揭示人物
- 对话要体现人物性格差异
- 伏笔要在前文埋设，后文回收
- 节奏张弛有度，紧张场景后安排缓冲
- 避免「上帝视角」泄露过多信息

## 常见问题
- 情节松散：缺少贯穿全文的主线冲突
- 人物扁平：缺少内在矛盾和成长弧线
- 高潮无力：前文铺垫不够，高潮显得突兀
